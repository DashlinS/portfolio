Task List

This project is a to-do list that has a few features once you list your to-dos! This was a fun project that focused on separating items and having them be effected one by one. I had a few ideas on the design, but settled on this simple image where it was clean and easily readable.

    Task list will allow you to
  • Write down each task one by one
  • Select each task and erase them one by one
  • Erase all the tasks at once
  • Not allow you to go over a certain amount of letters!!
